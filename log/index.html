Журнал

03.03.2025 — Презентация концепции настольной игры “Выход есть!”

На занятии по проектной деятельности была представлена первая отчётная презентация по настольной игре “Выход есть!”. Разработкой концепции игры занимались Неткачев Даниил и Лендер Марк (группа 241–321). Были представлены правила игры, карточки ситуаций, набросок поля и др.

05.03.2025 - Разработка Локаций для игры “Safaty Madness”

Масленников Олег (241-335), начал продумывание локаций, эвентов и всяческой аналитики данных подаваемых в процессе игры, вроде подсказок и рекомендаций

07.03.2025 — Старт разработки компьютерной игры “Safety Madness”

Комаров Егор (241–335), совместно с другими членами команды, создал репозиторий проекта на GitHub и начал писать код. Также Егор разработал меню и начал реализацию игровых механик.

24.03.2025 — Презентация карточек ситуаций для настольной игры

Лендер Марк (241–321), совместно с командой, представил новый дизайн игровых карточек. Он занимался оформлением и наполнением карточек, посвящённых критическим ситуациям.

14.04.2025 — Промежуточная презентация по “Safety Madness”

Неткачев Даниил (241–321) представил отчёт, в котором показал, что уже готово в игре: сюжет, локации, таблицы с правилами безопасности и другое.

17.04.2025 — Готовы анимации для компьютерной игры

После отрисовки Лендер Марком (241–321) были представлены готовые анимации персонажей для компьютерной игры “Safety Madness”.

21.04.2025 — Демоверсия почти готова!

Комаров Егор (241–335) представил сборку с анимациями и первой локацией, наполненной событиями. Работа над первым уровнем почти завершена.

28.04.2025 - Провеение апробации
